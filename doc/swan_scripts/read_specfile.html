PURPOSE

Read the ascii-based spectral distribution output from SWAN for single station

DESCRIPTION 

 Read the ascii-based spectral distribution output from SWAN for single station 

 DESCRIPTION:
   function [specstruct] = read_specfile(specfile,plotall) 

 INPUT
