 function fieldout = nodes2elems(fieldin, Mobj)  

 DESCRIPTION:
    Transfer a field from vertices (nodes) to elements

 INPUT
    Mobj         = Matlab mesh object
    fieldin      = vertex-based field

 OUTPUT:
    fieldout = element-based field

 EXAMPLE USAGE
    f = smoothfield(fv, Mobj)
