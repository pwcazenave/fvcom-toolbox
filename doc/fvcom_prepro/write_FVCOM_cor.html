 function write_FVCOM_cor(Mobj,filename)

 DESCRIPTION:
    Generate an ascii FVCOM 3.x format coriolis from Mesh object

 INPUT 
   Mobj     = Mesh object
   filename = FVCOM coriolis file name

 OUTPUT:
    FVCOM coriolis file: filename

 EXAMPLE USAGE
    write_FVCOM_cor(Mobj, 'tst_cor.dat')
